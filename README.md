TasteMap 🍽️

TasteMap is a simple and social restaurant discovery app that helps users save, rate, and review restaurants all in one place. Whether you’re tracking your favorite burger spot, discovering hidden cafés, or sharing food experiences with friends, TasteMap makes exploring food easier and more fun.

Features
⭐ Rate restaurants with custom scores
📝 Write and edit restaurant reviews
📍 Save favorite places on an interactive map
🔎 Search restaurants by name, cuisine, or location
📸 Upload food and restaurant photos
❤️ Bookmark favorite restaurants
👥 Follow friends and see their reviews
🏷️ Add tags like “Best Pizza” or “Date Night”
🌙 Dark mode support
📱 Responsive design for mobile and desktop
Tech Stack
Frontend: React / Android / iOS
Backend: Spring Boot / Node.js
Database: PostgreSQL / MySQL
Maps API: Google Maps or OpenStreetMap
